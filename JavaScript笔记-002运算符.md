# 运算符

|    类型    |      符号      |              含义               |
| :--------: | :------------: | :-----------------------------: |
| 算数运算符 |       +        |              加号               |
|            |       -        |              减号               |
|            |       \*       |              乘号               |
|            |       /        |              除号               |
|            |       %        |              取余               |
|            |       ++       |              自增               |
|            |       --       |              自减               |
| 赋值运算符 |       +=       |             加等于              |
|            |       -=       |             减等于              |
|            |      \*=       |             乘等于              |
|            |       /=       |             除等于              |
|            |       %=       |            取余等于             |
| 比较运算符 |       >        |              大于               |
|            |       <        |              小于               |
|            |       >=       |            大于等于             |
|            |       <=       |            小于等于             |
|            |       ==       |           比较值相等            |
|            |       !=       |            值不相等             |
|            |      ===       |        比较值和类型相等         |
|            |      !==       |         值和类型不相等          |
| 逻辑运算符 |      \|\|      |               或                |
|            |       &&       |               且                |
|            |       !        |               非                |
| 三元运算符 | exp1?exp2:exp3 | 满足 exp1，则取 exp2，否则 exp3 |
| 其他运算符 |      void      |        表达式无返回结果         |
|            |     typeof     |       返回数据对应的类型        |
